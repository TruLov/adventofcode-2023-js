# adventofcode 2023 JS

```bash
# link cli to global node_modules with
npm run link

# run cli with
aoc --version 

# get help
aoc --help
aoc bench --help

# test and solve part 01 with
# (solve will always look for `input.txt` if not specified via `-f`)
aoc solve 1 --f test.txt
aoc solve 1

# run micro benchmarks 1000 times
aoc bench 02 -i 1000 -t micro
# iqr
aoc bench 02 -i 1000
```
